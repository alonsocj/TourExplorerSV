## Estructura de directorio

La estructura de directorio de un modulo es la siguiente:

```
  modules/
    modulo/
      Controllers/
        NameController.php
      Models/
        NameModel.php
      Views/
        index.tpl
      Routes/
        NameRoutes.php
      routes.php
```
